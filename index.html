A feather-like text editor March 7, 2024

Text editors nowadays (the fancy ones like vscode, sublime, and whatever is the new cool editor in the block) all have the same issue: they can't seem to use little resources, they're all amazingly heavy. So, I decided to ditch out electron and all the weird frameworks for a code editor written from scratch inspired by no other than Netscape Composer (at least.. the name Composer)

I shall present.. the new.. Irie Composer!

yeah.. not finished so its not that worth to already start comparing it to the modern ones.. but hey, its going there. UI was written from scratch with OpenGL, and text rendering being done using Multi Signed Distance Fields (MSDF) But not to worry, it sounds heavy but its really efficient, quite more than mesh based font rendering. Fonts used are the same as the websites, with the exception the editor uses MPLUS Code instead of IBM Plex Mono.

Anyway, that's enough for today. I'll keep posting updates on this project here and eventually give its own page once it becomes usable.
